# CRONÔMETRO DIGITAL

## Sobre

<p>Cronômetro digital simples, que permite ao usuário controlar
os segundos, minutos e até horas de uma determinada atividade.</p>

## Features

- [x] Inciar cronômetro.
- [x] Pausar cronômetro.
- [x] Zerar cronômetro.
